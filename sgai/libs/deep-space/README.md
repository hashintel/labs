# deep-space

This is a stub for validating the best figma/code integration for the new HASH design system. 

Based on the `tsdown` React starter including vite, rolldown and the react-compiler.

## Development

- Install dependencies:

```bash
npm install
```

- Run the playground:

```bash
npm run play
```

- Run the unit tests:

```bash
npm run test
```

- Build the library:

```bash
npm run build
```
